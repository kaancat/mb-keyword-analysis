# mb-keyword-analysis

Claude Code plugin for Google Ads keyword analysis workflow with enforced phase gates.

## Installation

```bash
/plugin install mb-keyword-analysis@mb-plugins
```

## Usage

```bash
/keyword-analysis
```

## Features

- 6-phase structured workflow
- Decision trees for match types, budgets, campaign structure
- Phase gate validation hooks
- Golden examples and templates

## Development

Clone to your preferred location:
```bash
git clone https://github.com/kaancat/mb-keyword-analysis.git ~/Development/mb-keyword-analysis
```

Edit, bump version in `.claude-plugin/plugin.json`, push, then update marketplace version.
